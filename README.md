# Academic Homepage (Mayday 25th Anniversary Theme)

This is an academic personal homepage template based on the color scheme of Mayday's 25th Anniversary. The page is built with simple HTML and CSS, requiring no complex build tools, making it easy to modify and deploy.

- **Color Palette**: Uses Mayday's classic five-color dots (Yellow, Orange, Pink, Blue, Green) plus Watermelon Red.
- **Elements**:
  - The five "dots" at the top represent the five members of Mayday, with simple CSS geometric shapes (sunglasses, eyes, etc.).
  - The "fruit" elements at the bottom echo the fresh and lively cartoon style.
  - **Fonts**: Google Fonts `Inter` (Body) and `Patrick Hand` (Handwritten Title) are used to create an academic yet lively atmosphere.

2. **Add Content**:
   - **About**: Modify the self-introduction text in `<section id="about">`.
   - **Education**: Add your degree information in the `<ul>` list within `<section id="education">`.
   - **Research**: List your research directions and keywords in `<section id="research">`.
   - **Publications**: Copy the `<div class="pub">...</div>` block in `<section id="publications">` to add new papers.
   - **Teaching** and **Awards**: Similarly, modify the list content in the corresponding `<section>`.

3. **Links & Files**:
   - Name your CV (Resume) PDF file as `cv.pdf` and place it in the same directory, or modify the link address in `<a href="cv.pdf">`.
   - Replace `mailto:you@university.edu` with your actual email address.
   - Replace Google Scholar, GitHub, etc., links with your homepage addresses.
